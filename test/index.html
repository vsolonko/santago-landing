Робот VIKA

Подберем сотрудников для вас. VIKA найдет сотрудников, обзвонит их и приведет к Вам.

Поиск кандидата с помощью VIKA

Если резюме соискателя есть в интернете, VIKA его найдет.

При поиске на всех работных сайтах, VIKA удаляет все совпадения. Вам не стоит беспокоиться о том, что вы позвоните одному кандидату дважды.

Робот VIKA справиться с поиском по различных вакансиям. Найдет инженера с узкой спецификацией, где бы не было его резюме, а также грузчика на склад.

  • Функциональность (ищет резюме по всему интернету, по заданным критериям заказчика)
  • Интерактивность (гибкость критериев поиска)
  • Скорость (более 100 звонков в минуту обновременно)
  • Гарантия (вы платите только за реальных кандидатов)

Алгоритм работы VIKA

Шаг 1

Клиент связывается с нашим менеджером. Менеджер принимает заявку по вакансии Клиента, учитывая все критерии поиска, помогает подобрать оптимальный тариф.

Шаг 2

Клиенту на согласование отправляется test запись по его вакансии. На данном этапе, Клиент вправе, по необходимости, внести поправки или корректировки, а если таких нет – утвердить полученную информацию.

Аудио VIKA - это демонстрационные возможности.

Шаг 3

VIKA автоматически производит выборку по заявленным требованиям. Начинает обзванивать всех найденных соискателей. Скорость обзвона – 100 чел/к – за 1 минуту.

Шаг 4

Клиенту отправляется таблица с данными соискателей, готовых работать на предложенных условиях. Таблица содержит Ф.И.О соискателя, его контактные данные.
